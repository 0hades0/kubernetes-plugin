Specifies a shell which will run inside the container and process requests from Jenkins to run executables, set environment variables, and similar infrastructure.

This does not affect the shell used to run user code, such as sh steps. To run your scripts with a specific shell, use an interpreter line: 


sh '''#!/bin/bash
for x in {0..9}; do echo x; done
'''

Defaults to sh, which should be in $PATH. Should not generally be overridden.
